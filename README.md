### running script

1. 依赖

   + python3
   + pip3
   + you_get
   + bs4

2. 执行

   ```
   python3 request_cxk.py {查询关键字} {需要下载到的页数}
   ```

   ​	`eg: python3 request_cxk.py cxk 1` *即搜索关键字cxk， 只要下载查询出来的1页的视频* 

   ​	`{需要下载到的页数}可以不用填写，默认为搜索出来的总的页数`
